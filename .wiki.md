# Project Summary
The OverDryv project is a multi-tenant application designed to streamline vehicle service management. It enables service providers to manage customer profiles, estimates, work orders, and invoicing efficiently. The platform aims to enhance user experience through a robust dashboard, facilitating seamless interactions between service providers and their customers.

# Project Module Description
- **Admin Dashboard**: Central hub for service management, including customer, vehicle, and work order management.
- **Customer Portal**: Interface for customers to view service history and manage their vehicles.
- **Estimates and Invoices**: Tools for generating and managing service estimates and invoices.
- **Authentication**: Secure user authentication and profile management.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL)
- **Build Tools**: Vite, ESLint

# Usage
1. **Install Dependencies**: Run `npm install` to install all required packages.
2. **Build the Project**: Use `npm run build` to build the application for production.
3. **Run the Application**: Execute `npm start` to start the development server.
