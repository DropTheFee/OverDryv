# Demo Data Import Instructions

## How to Import into Supabase

1. **Go to your Supabase Dashboard**: https://supabase.com/dashboard
2. **Select your OverDryv project** (qbhimvjkmsulmjmhduzh)
3. **Click "Table Editor"** in left sidebar

### Import Customers
1. Select the **"profiles"** table
2. Click **"Insert"** → **"Import data from CSV"**
3. Upload `customers.csv`
4. Map columns:
   - email → email
   - first_name → first_name
   - last_name → last_name
   - phone → phone
   - role → role
5. Click **"Import"**

### Import Vehicles (After customers are imported)
1. First, you need customer IDs from the profiles table
2. Select the **"vehicles"** table
3. Click **"Insert"** → **"Insert row"** (easier for now with relationships)
4. Or modify `vehicles.csv` with actual customer UUIDs and import

### Notes
- The `id` field will be auto-generated by Supabase
- `created_at` and `updated_at` will be set automatically
- Make sure customers exist before adding their vehicles
- For demo user login: demo@overdryv.io / Demo123! (already exists)
